# Product Specification

## Vision

Build a calm, practical Soroban learning dojo on GitHub Pages with attempt-first exercises, local-first progress, and inclusive paths for children and adults.

## Core promise

- learn step by step
- practice immediately
- attempt first, then check
- track progress locally
- stay motivated without pressure

## Target users

- children learners
- adult self-learners
- parents and teachers
- contributors expanding open lessons and exercises

## Product principles

- static-first
- privacy-first
- authored curriculum first
- practical over flashy
- Japanese dojo tone: calm, respectful, disciplined

## MVP goals

1. Publish a GitHub Pages-ready Astro site.
2. Provide lessons and exercises from foundations into beginner arithmetic.
3. Support a one-at-a-time practice flow with answer reveal after attempt.
4. Store progress in the browser without accounts.
5. Include a support page for donations and contribution.
