Each rod keeps its own digit, but its position changes the meaning.

## Step by step for 24

1. On the tens rod, set **2** lower beads to the beam. That means **20**.
2. On the ones rod, set **4** lower beads to the beam. That means **4**.
3. Read left to right: **20 + 4 = 24**.

## Common mistake

Reading each rod correctly but forgetting which rod is tens and which rod is ones.
