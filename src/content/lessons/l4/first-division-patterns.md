Early division is easier when you see how a number can be separated evenly.

## Example: 12 ÷ 3

1. Start from **12**.
2. Split the value into **3 equal groups**.
3. Each group holds **4**.
4. Check the result by thinking **4 + 4 + 4 = 12**.
