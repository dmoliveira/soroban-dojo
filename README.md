# Soroban Dojo

A GitHub Pages-ready Astro site for learning Soroban through calm, practical lessons and attempt-first exercises.

## Stack

- Astro static export
- Markdown content collections
- Local-first progress tracking in the browser

## Scripts

- `npm install`
- `npm run dev`
- `npm run build`
- `npm run check`

## Structure

- `src/content/` source lessons, exercises, and references
- `src/pages/` public pages
- `src/components/` reusable UI blocks
- `docs/specs/` product and implementation specifications
